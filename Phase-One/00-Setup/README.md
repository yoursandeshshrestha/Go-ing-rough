# Phase One: Setup

## 00-setup

### Steps:

1. Installed Go.
2. Verified installation with `go version`.
3. Created a workspace for Go projects.
4. Initialized a Go module with `go mod init Go-ing-rough`.
5. Wrote and ran my first Go program: `Hello, Go!`

---

### First Go Program

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, Go!")
}
```
